# ログイン

<Image src="/assets/login/login.png" alt="Login" height="500" />

アプリケーションを使用するためにはログインが必要です。<br></br>
ログインには、次の情報が必要になります。

- サーバーのアドレス
- ユーザーID
- パスワード

<Info>これらは、予め管理者権限を持ったユーザーに作成してもらう必要があります。</Info>

## アドレスの入力

アドレスはご利用の施設によって異なります。

## ユーザーIDとパスワードの入力

ログインに使用するアカウントのユーザーIDとパスワードを入力します。

##  ログインする

入力が完了したら**ログイン**ボタンをタップします。
