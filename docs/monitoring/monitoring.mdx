# モニタリング

この画面では患者の現在の状態をリアルタイムで表示することができます。<br></br>

<Image src="/assets/monitoring/monitoring.png" alt="Monitoring" height="500" />

表示されているものは以下の通りです。

1. 患者の現在の状態を示すアイコン
2. 患者の名前と部屋
3. 患者が利用しているベッドの設定

## 患者の詳細を表示する

患者の詳細情報を表示したい場合は、その患者をタップします。<br></br>
画面下部からボトムシートが表示されます。<br></br>
ボトムシートには、患者の詳細ページを開くボタンや、患者が利用しているベッド等を操作するためのボタンが表示されています。

<Image src="/assets/monitoring/monitoring-bottom-sheet.png" alt="Monitoring-Bottom-Sheet" height="500" />


## アラートON

患者のアラートがONになると、患者のアイコンが変化します。<br></br>

<Image src="/assets/monitoring/monitoring-alert-on.png" alt="Monitoring-Alert-ON" height="500" />


また、右下にベルのアイコンが表示され、アラートがONになっている患者のリストを表示することができます。<br></br>
患者の状態を確認し、アラートをOFFにしたい場合は、「確認OK」ボタンをタップすることで、アラートをOFFにすることができます。

<Image src="/assets/monitoring/monitoring-alert-drawer.png" alt="Monitoring-Alert-Drawer" height="500" />
