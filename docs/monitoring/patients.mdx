# モニタリング

この画面では利用者の現在の状態をリアルタイムで表示して、現在の利用者の状態やアラートなどを確認することができます。<br></br>

<Image src="/assets/monitoring/monitoring.png" alt="Monitoring" height="500" />

表示されているものは以下の通りです。

1. 利用者の現在の状態を示すアイコン
2. 利用者の名前と部屋
3. 利用者が利用しているベッドの設定

## 利用者の詳細を表示する

利用者の詳細情報を表示したい場合は、その利用者をタップします。<br></br>
画面中央にダイアログが表示されます。<br></br>
ダイアログには、利用者の詳細ページを開くボタンや、利用者が利用しているベッド等を操作するためのボタンが表示されています。

<Image src="/assets/monitoring/monitoring-dialog.png" alt="Monitoring-Dialog" height="500" />


## アラートON

利用者のアラートがONになると、利用者のアイコンが変化します。<br></br>

<Image src="/assets/monitoring/monitoring-alert-on.png" alt="Monitoring-Alert-ON" height="500" />


また、右下にベルのアイコンが表示され、アラートがONになっている利用者のリストを表示することができます。<br></br>
利用者の状態を確認し、アラートをOFFにしたい場合は、「確認OK」ボタンをタップすることで、アラートをOFFにすることができます。

<Image src="/assets/monitoring/monitoring-alert-drawer.png" alt="Monitoring-Alert-Drawer" height="500" />
