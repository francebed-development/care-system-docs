# テーブル

レポート画面の右上のアイコンをタップすることで、患者のステータスの履歴をテーブル表示することができます。

<Image src="/assets/report/report-table.png" alt="Report-Table" height="500" />

## CSVファイルに出力する

右上のアイコンをタップして、患者のステータスの履歴をCSVファイルに出力することができます。<br></br>
CSVファイルに出力することで、市販の表計算ソフトを使用して表示することも可能です。<br></br>

<Image src="/assets/report/report-create-csv.png" alt="Report-Create-CSV" height="500" />
