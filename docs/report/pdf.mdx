# PDF

レポートをPDF形式で出力することができます。<br></br>
レポート画面の右上のアイコンをタップすることで、患者のグラフを表示することができます。

## PDFファイルに出力する

右上のアイコンをタップして、患者のグラフをPDFファイルに出力することができます。<br></br>
