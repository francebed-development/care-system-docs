# アラートの設定

この画面では、アラートに関する設定を変更することができます。

<Warning>登録されている全ての利用者に適用されます。</Warning>

<Image src="/assets/setting/alert-setting/alert-setting.png" alt="Alert-Setting" height="500" />

## 1.アラートの再通知を設定する

アラートが発生してから、一定時間経過してもご利用者様の確認が行われない場合に、アラートを再通知することができます。

<Image src="/assets/setting/alert-setting/remind-alert.png" alt="Remind-Alert" height="500" />

アラートを再通知させるために、まず「アラートの再通知を設定する」を**ON**にします。
ONにすると以下のような設定フォームが表示されます。<br></br>

<Image src="/assets/setting/alert-setting/remind-alert-input-form.png" alt="Remind-Alert-Input-Form" height="500" />

再通知の間隔は5分〜60分の間から選択することができます。

設定内容に問題がなければ、画面下部の保存ボタンをタップします。
<br></br>

## 2.アラートをOFFにする時間帯を設定する

アラートをOFFにする時間帯を設定することができます。

<Image src="/assets/setting/alert-setting/alert-off-setting.png" alt="Alert-Off-Setting" height="500" />

アラートを鳴らさない時間を設定するために、まず「アラートをOFFにする時間帯を設定する」を**ON**にします。
ONにすると以下のような設定フォームが表示されます。<br></br>

<Image src="/assets/setting/alert-setting/alert-off-setting-input-form.png" alt="Alert-Off-Setting-Input-Form" height="500" />

アラートをOFFにしたい時間の「開始時間」と「終了時間」をそれぞれ選択します。
画面下部にアラートをOFFにする時間帯のグラフが表示され、設定内容を表示ます。
グラフの配色は以下のようになっています。

- ピンクがアラートON
- グレーがアラートOFF

設定内容に問題がなければ、画面下部の保存ボタンをタップします。
