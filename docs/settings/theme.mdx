# テーマの変更

アプリケーションのテーマを変更することができます。<br></br>

<Image zoom src="/assets/setting/setting-personal.png" alt="Setting-Personal" height="500" />

「テーマを変更する」をタップするとダイアログが表示され、次の4つの選択肢があります。

<Image zoom src="/assets/setting/theme/theme-dialog.png" alt="Theme" height="500" />

- システム
    - 端末の設定情報から使用するテーマを決定します。
- ライト
    - 明るいテーマを使用します。
- ダーク
    - 暗いテーマを使用します。
- ピュアダーク
    - ダークよりもさらに暗いテーマを使用します。

<Info>テーマの設定は、端末ごとに行うことができるので、お好みで設定してください。</Info>
