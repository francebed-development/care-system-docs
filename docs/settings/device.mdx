# ご利用中の端末一覧

この画面では、システムを利用している端末の一覧を表示しています。

<Image src="/assets/setting/device/device.png" alt="Device" height="500" />

<Info>一覧に表示されている端末にサーバーからプッシュ通知が送信されます。</Info>

## 端末情報の削除

ゴミ箱のアイコンをタップすると、端末情報の削除を確認するダイアログが表示されます。<br></br>
端末情報を削除すると、その端末にプッシュ通知が送信されなくなります。<br></br>

<Image src="/assets/setting/device/device-delete.png" alt="Device-Delete" height="500" />

<Info>再登録する場合は、登録したい端末で、一度ログアウトして、再度ログインしなおしてください。</Info>
