# センサーの設定変更

この画面では、ご利用中のベッドの設定を変更することができます。

<Warning>アプリケーションからベッドの設定を変更するためには、ベッドの通信方法を**UDP**にする必要があります。</Warning>

## 個別に設定する

1. センサーを個別に設定する場合は、設定を変更したいベッドを利用している患者をタップします。
2. アラームモードと感度を選択して、下部にある保存ボタンをタップします。

## 一括設定

表示されているベッドの設定を一括で設定することができます。<br></br>

1. 画面右下の「**一括設定**」ボタンをタップします。
2. 設定を変更したいベッドを利用している患者にチェックマークを入れます。
3. 画面右下の「**設定する**」ボタンをタップします。
4. アラームモードと感度を選択して、下部にある保存ボタンをタップします。

<Info>稀に設定の反映に失敗することがあります。その場合は、再度試してください。</Info>
