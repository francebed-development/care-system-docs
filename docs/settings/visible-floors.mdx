# 表示するフロアの選択

この画面では、アプリケーションに表示するフロアを編集することができます。

<Image src="/assets/setting/visible-floor/visible-floor.png" alt="Visible-Floor" height="500" />

表示・非表示を切り替えたいフロアの右のスイッチのON・OFFを切り替えます。<br></br>
設定ができたら、下部の保存ボタンをタップします。

<Info>この設定はログイン中のアカウントに紐づけられます。</Info>

<Info>
表示をONにしているフロアにいる患者のアラート(プッシュ通知)を受信します。<br></br>
※「**ALL**」がONになっている場合は全ての患者のプッシュ通知を受信します。
</Info>
