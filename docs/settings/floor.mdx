# フロアの編集

この画面では、ご利用の施設のフロアを編集することができます。

<Image src="/assets/setting/floor/floor.png" alt="Floor" height="500" />

<Warning>同じ名前のフロアを複数登録することはできません。</Warning>

## 新規登録

ご利用のフロアを新規登録する場合は、画面右上の＋アイコンをタップします。<br></br>
新規登録画面に遷移するので、登録するフロアの名前を入力し、下部にある保存ボタンをタップします。

<Image src="/assets/setting/floor/floor-new.png" alt="Floor-New" height="500" />

<Info>フロアの名前は「1F」「2F」などスタッフが識別しやすい名前にすることを推奨します。</Info>

## 登録内容の変更

すでに登録されているフロアの情報を変更したい場合は、変更したいフロアをタップします。<br></br>
編集画面に遷移するので、変更内容を入力して、下部にある保存ボタンをタップします。

<Image src="/assets/setting/floor/floor-edit.png" alt="Floor-Edit" height="500" />

## 削除

登録されているフロアを削除したい場合は、削除したいフロアをタップします。<br></br>
編集画面に遷移するので、画面右上のゴミ箱のアイコンをタップします。<br></br>
削除を確認するダイアログが表示されるので、削除する場合は、「削除」のボタンをタップしてください。

<Image src="/assets/setting/floor/floor-delete.png" alt="Floor-Delete" height="500" />

<Error>削除してしまうと、復元することはできません。</Error>
