# プッシュ通知の設定

「プッシュ通知の設定」をタップすると、端末の設定画面を開くことができます。<br></br>

<Image src="/assets/setting/notification/notification-setting.png" alt="Notification-Setting" height="500" />

プッシュ通知を受信したい場合は、プッシュ通知を許可してください。受信したくない場合は、プッシュ通知を拒否することができます。

<Image src="/assets/setting/notification/notification-setting-detail.png" alt="Notification-Setting-Detail" height="500" />
