# アラートログ

この画面では、患者のアラートの履歴を確認することができます。<br></br>
画面上部のタブをタップすることで、表示したいアラートログをフロアごとに表示することができます。<br></br>

<Image src="/assets/alert-log/alert-log.png" alt="Alert-Log" height="500" />

アラートログの詳細を表示したい場合は、表示したいアラートログをタップすると、詳細が記載されたダイアログが表示されます。

<Image src="/assets/alert-log/alert-log-dialog.png" alt="Alert-Log-Dialog" height="500" />
